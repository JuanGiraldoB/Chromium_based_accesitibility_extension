Aumentar Texto

Este botón te permitirá aumentar el tamaño de todos los textos en cualquier sitio web.

Disminuir Texto

Este botón te permitirá disminuir el tamaño de todos los textos en cualquier sitio web.

Aumentar Interlineado

Este botón te permitirá aumentar el espacio vertical que existe entre todo el contenido de un sitio web.

Disminuir Interlineado

Este botón te permitirá disminuir el espacio vertical que existe entre todo el contenido de un sitio web.

Tipografía - Menú

En este menú encontraras las opciones para cambiar el tipo de fuente, cambiar los colores de párrafos, enlaces y títulos, y además, podrás activar o desactivar el inter letrado.

Narrador

Este botón te narrará cualquier texto sobre el que hagas click, este texto será subrayado en el color que hayas escogido.

Pausa

Este botón pausará la narración del texto que este subrayado por el botón Narrador.

Contraste - Menú

En este menú encontraras dos opciones, contrastes predeterminados que cambiaran tanto el color de fondo del sitio web como el color de los textos y la segunda opción te permitirá cambiar el color de fondo de manera personalizada.

Cargar Ajustes

Este botón cargará los ajustes que hayas realizado. Es ideal para cargar tus preferencias dentro de cualquier sitio web.

Configuración - Menú

En este menú encontraras una serie de checkboxes que te permitirán desactivar/activar las distintas opciones que quieres que se visualicen dentro de la herramienta. También podrás cambiar la velocidad del narrador y el color de resaltado de este.
